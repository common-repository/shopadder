== Installation ==

<h3>Automatic installation</h3>

1. Login to your WordPress admin panel, navigate to the Plugins menu and click Add New.
1. Search for 'ShopAdder' and click at Install now.	ShopAdder will then be automatically downloaded and installed. 
1. To get a quick impression how it works: just type <strong>{shopaddertest}</strong> now, anywhere in your editor to see a demo product.

<h3>Manual installation</h3>

1. Upload `shopadder.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. To get a quick impression how it works: just type <strong>{shopaddertest}</strong> now, anywhere in your editor to see a demo product.

<h3>What's next?</h3>
 
1. Create a free account at [ShopAdder](http://www.shopadder.com/index.php?do=free "Create a FREE account").
1. Go back to WordPress, click SETTINGS in the left menu bar and select ShopAdder. Then enter the Shop ID and the Server ID in the corresponding fields in the ShopAdder Settings section.";
<br>*Or leave as is to use the demo shop.*
1. Start adding products.<br>*For example: type {shopaddertest} anywhere in your site to see a demo product from your own ShopAdder shop.*

== Frequently Asked Questions ==

= I do not see any changes =

If you entered your own shop ID and still get the shopaddertest checkout, please
re-publish the WP page that you are viewing. In WP go to the Post/Page, click Edit and click the update button.

Please use the feedback form at [ShopAdder](http://www.shopadder.com/ "Ask a question"), 
or use the WordPress options to contact us.
